# Customer-Data-Analysis-SQL
This project analyzes customer data using SQL to gain insights into customer behavior , sales trends, and business performance  

## Tools Used
- MySQL
- SQL

## Objectives 
- Analyze customer information
- Identify top customer
- Calculate total sales
- Find customer purches patterns
- Generate business insights

 ## SQL Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- JOINS
- Views
